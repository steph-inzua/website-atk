#DIGITALMENTE

NUEVA FORMA
DE COMUNICACIÓN

El BCP presenta #Digitalmente como una nueva plataforma de comunicación. Esta web tiene como objetivo reflejar el compromiso del banco en seguir desarrollando soluciones innovadoras que permitan a sus clientes tener y llevar al BCP con ellos toda la semana, las 24 horas al día, cuando y donde quieran , para que sus clientes tengan una mejor experiencia de manera más simple.
#Digitalmente se crea para enseñar a usar las diferentes soluciones digitales que el banco presente a disposición de sus clientes; como también para comunicar las innovaciones del banco.

SERVICIOS EMPLEADOS

HTML5 Website, SEO

LLEGADA
INNOVADORA

A través de la agencia de publicidad TBWA, se presenta la oportunidad de presentar las distintas soluciones innovadoras del BCP a lo largo de los años. Al ser una gran cantidad de soluciones y aprovechando la temática "sobre el espacio", se utiliza una navegación one-page, que permite una navegación fluida para el usuario sin necesidad de cargar nuevas páginas.
Para aligerar la carga de la página se evita el uso de un background con fotografías y se realiza una animación HTML5 / CSS3 simulando el espacio y las estrellas flotantes.

TECNOLOGÍA UTILIZADA

HTML5, CSS3, Parallax Scrolling
